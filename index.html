Mp3 player

The mp3 player project is a fully functional music listening environment in which the user can browse through all the songs he has added to the application. The mp3 features are browsing between songs, pausing and continuing the song, as well as live lyrics (the user must add the file). Also, for each song the name of the artist and the song is displayed. The UI is simple and very easy to use it!

Java JavaFx Scene Builder

Rent app

The application I created is for renting houses, cars, boats or even jets. The user can browse through the various available ads and, depending on his wishes, rent a service within the time available. The total cost is calculated dynamically depending on the cost of the service. When the user makes a reservation for a service, a confirmation email is sent to the corresponding account they have provided. Each user has their own account whose data is stored in a remote firestore database and authentication is required for their connection. The ability to change the connection password is also provided.

Java Kotlin Android Studio

Simple e-shop

The e-shop I created is a very simple project. There are some categories for the user to browse and when they select one of them, there are corresponding products in each of them, with a brief description of the product as well as its price. Whatever the user chooses is collected in his cart and he can make his purchase.
